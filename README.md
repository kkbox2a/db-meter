# 專業分貝監測儀 (Web Decibel Meter)

這是一個基於網頁標準 Web Audio API 開發的即時環境音量（分貝）偵測器。
透過響應式設計與 PWA 技術，不僅可以在電腦瀏覽器上使用，更能完美支援 iPhone (iOS) 與 Android 手機，提供宛如原生 App 般的全螢幕沉浸式體驗。

## ✨ 功能特色

* **即時音量偵測**：透過麥克風即時收音，將振幅轉換為分貝值 (dB)。
* **動態歷史圖表**：整合 Chart.js，流暢繪製過去一段時間的音量起伏曲線。
* **視覺化警告系統**：根據音量大小自動變換數字與圖表顏色（例如：超過 80 dB 顯示紅色警告）。
* **峰值紀錄**：自動追蹤並記錄單次監測過程中的最高分貝值。
* **PWA 支援 (漸進式網頁應用)**：支援「加入主畫面」功能，可安裝於手機桌面，支援離線快取與全螢幕顯示。

## 🛠️ 本地開發與測試

由於瀏覽器安全性限制（特別是 iOS Safari），存取麥克風的 `getUserMedia` API 必須在 **HTTPS** 環境下執行。若要在本地端測試，請依下列步驟建立臨時 HTTPS 伺服器：

1. 確保已安裝 Python 環境。
